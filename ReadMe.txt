Windows 10 Operating System
Visual Studio Community 2017
Written in C++

Takes input from "Input.txt" which has 4 sections.

Entries under [ADD] should represent one vehicle per line to add to inventory
   Entries have format "VIN,Manufacturer,Country,Model,Year,Color"

Entries under [DELETE] should be whole VIN values to remove from inventory

There should be one line under [YEAR] to output a list of only vehicles of that year

There should be one line under [MANUFACTURER] to output a list of only vehicles of that manufacturer
